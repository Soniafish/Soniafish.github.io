填寫說明

【單位法令遵循主管異動】

  • 1.單位申請表由新調/新任單位提出申請，並經單位主管核定。
  • 2.單位法令遵循主管資格：
    • 一、職級為科主管級以上。
    • 二、熟稔該單位業務及相關法令，並預備協助推動該單位法令遵循業務之能力者。
    • 三、應於就任前或就任後半年內具備下列資格條件之一：
      • (一)曾任金融機構法令遵循人員或主管，合計滿五年者。
      • (二)參加主管機關認定機構所舉辦三十小時以上課程，並經考試及格取得結業證書。
      • (三)依本公司自行擬定之具體訓練計畫，參加富邦金控或公司自行舉辦三十小時以上相關訓練課程及測驗，足證已具備熟知單位所需定之相關能力。
  • 3.單位法令遵循主管異動時，應於交接後三日內提出申請，逾期申請請洽法令遵循部辦理。
